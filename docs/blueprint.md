# **App Name**: DoxaCare AI

## Core Features:

- AI Chatbot: AI-powered chatbot to answer health-related questions and provide advice, using reasoning to guide the user.
- Language Toggle: Multilingual support for English and Krio
- Tab Navigation: Tab-based navigation to switch between different app sections.
- Health Alerts: Display of current health alerts and notifications from reliable sources.
- Clinic Finder: Directory of nearby clinics, searchable by location.
- Emergency Assistance: Emergency contact information and a modal for immediate assistance.
- Health Resources: Collection of links to reliable and relevant health information and reports.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke health and nature.
- Background color: Light green (#F0FDF4), a very light tint of the primary to suggest a clean, natural and welcoming environment
- Accent color: Yellow-orange (#FDD835) to highlight calls to action and important information.
- Body and headline font: 'PT Sans' (sans-serif) for a balance of modern look with a little warmth or personality
- Code font: 'Source Code Pro' (monospace) for displaying code snippets
- Use Font Awesome icons throughout the interface to add visual cues and enhance usability.
- Subtle animations and transitions to provide feedback and enhance the user experience.
